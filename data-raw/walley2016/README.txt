This directory contains data published in Walley 2016 PMID: 27540173

File:
GSE50191_FPKM.tsv
Source:
ftp://ftp.ncbi.nlm.nih.gov/geo/series/GSE50nnn/GSE50191/suppl/GSE50191%5FFPKM%2Etsv%2Egz
Contents:
This is a matrix of maize gene (v2) expression using Illumina HiSeq 2500 over 68 tissues summarized in FPKM values, but use a "tracking id" to designate tissues samples.  Mapped to b73 v2 FGS.

File:
Tracking_ids.csv
Source:
This file was hand-made, but was based on the S1 table for the paper.
Contents:
A map between the tracking ids in GSE50191 file, the name of the sample, and its replicate.
